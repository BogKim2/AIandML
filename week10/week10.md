## 🎯 들어가기: 전자기학이란?

### 전자기학의 4가지 기본 법칙

1. **가우스 법칙 (전기)**: ∇·E = ρ/ε₀
   - 전하가 전기장의 근원
   
2. **가우스 법칙 (자기)**: ∇·B = 0
   - 자기 홀극(monopole)은 존재하지 않음
   
3. **패러데이 법칙**: ∇×E = -∂B/∂t
   - 변하는 자기장이 전기장 생성
   
4. **앙페르-맥스웰 법칙**: ∇×B = μ₀J + μ₀ε₀∂E/∂t
   - 전류와 변하는 전기장이 자기장 생성

### 왜 시뮬레이션인가?

- 점전하
- 무한 평면

- 원통 대칭

**수치 시뮬레이션이 필요한 경우 (대부분!)**:

- 임의 형상의 도체
- 시간에 따라 변하는 장
- 비선형 매질

## 🔬 Program 01: 전기장 기초 (01_electric_field_basics.py)

### 물리적 배경

#### 쿨롱의 법칙
```
F = k * (q1*q2) / r²
```

여기서:
- `k = 8.99×10⁹ N·m²/C²` (쿨롱 상수)
- `q1, q2`: 전하량 (C)
- `r`: 전하 사이 거리 (m)

#### 전기장 정의
```
E = F/q = k*Q/r²
```

전기장은 **단위 전하가 받는 힘**입니다.

### 프로그램 실행
```bash
cd week10
python 01_electric_field_basics.py
```

### 출력 파일
- `outputs/01_single_charge_field.png`: 단일 점전하의 전기장
- `outputs/01_charge_comparison.png`: 양전하 vs 음전하

### 주요 개념

1. **벡터장 (Vector Field)**
   - 공간의 각 점에서 방향과 크기를 가짐
   - 화살표로 표현
   - 색상으로 크기 표현

2. **전기장의 성질**
   - 양전하: 밖으로 발산 (divergence)
   - 음전하: 안으로 수렴 (convergence)
   - 크기: `E ∝ 1/r²` (거리의 제곱에 반비례)

3. **시각화 방법**
   - **Quiver Plot**: 벡터 화살표
   - **Contour Plot**: 등고선 (크기)
   - **Color Map**: 배경 색상

### 물리적 통찰

**왜 1/r²인가?**
- 3차원 공간에서 구의 표면적: `A = 4πr²`
- 같은 "전기력선" 개수가 더 큰 구에 퍼짐
- → 밀도(= 세기)는 `1/r²`

**단위 확인**:
```
[E] = N/C = (kg·m/s²)/C = V/m
```

전기장의 단위는 `V/m`(볼트 퍼 미터)로도 표현됩니다!

## 🔬 Program 02: 전위 (02_electric_potential.py)

### 물리적 배경

#### 전위 (Electric Potential)
```
V = k*Q/r
```

전위는 **스칼라장** (방향 없음, 크기만 존재)입니다.

#### 전기장과 전위의 관계
```
E = -∇V = -(∂V/∂x, ∂V/∂y, ∂V/∂z)
```

전기장은 전위의 **기울기(gradient)**입니다!

### 주요 개념

1. **등전위선 (Equipotential Lines)**
   - 전위가 같은 점들을 연결한 선
   - 전기장에 **수직**
   - 전하를 이동시킬 때 **일이 필요 없음**

2. **전위와 일**
   ```
   W = q * ΔV
   ```
   전하를 이동시키는 일 = 전하량 × 전위차

3. **전위의 기준**
   - 무한대에서 0으로 정의
   - 상대적인 값만 의미 있음

### 물리적 통찰

**전기장 vs 전위**:

| 전기장 (E) | 전위 (V) |
|-----------|---------|
| 벡터 | 스칼라 |
| 방향 있음 | 방향 없음 |
| ∝ 1/r² | ∝ 1/r |
| 측정 어려움 | 측정 쉬움 (전압계) |

**왜 전위를 사용하나?**
- 계산이 간단 (벡터 합 대신 스칼라 합)
- 측정이 쉬움
- 에너지와 직접 연관

## 🔬 Program 03: 전기력선 (03_electric_field_lines.py)

### 물리적 배경

#### 전기력선 (Electric Field Lines)
- 전기장의 **방향**을 나타내는 곡선
- 양전하에서 **시작**, 음전하에서 **끝**
- 절대 **교차하지 않음**

#### 전기력선의 성질
1. 접선 방향 = 전기장 방향
2. 밀도 ∝ 전기장 세기
3. 등전위면에 수직

### 주요 개념

1. **쌍극자 (Dipole)**
   ```
   +q ←--d--→ -q
   ```
   - 가장 간단한 비대칭 전하 분포
   - 분자의 전하 분포 모델
   - 예: 물 분자 (H₂O)

2. **사극자 (Quadrupole)**
   ```
   +q  -2q  +q
   ```
   - 쌍극자보다 복잡
   - 멀리서 더 빠르게 감소 (∝ 1/r³)

3. **Streamplot**
   - Matplotlib 함수
   - 벡터장의 유선 시각화
   - 화살표로 방향 표시

### 물리적 통찰

**쌍극자의 중요성**:
- 대부분의 분자는 쌍극자 모멘트를 가짐
- 물, 암모니아, HCl 등
- 분자간 상호작용의 근원
- Van der Waals 힘

**전기력선 개수**:
- 가우스 법칙: `∮ E·dA = Q/ε₀`
- 전하 `Q`에서 나오는 전기력선 개수 ∝ `Q`

## 🔬 Program 04: 자기장 기초 (04_magnetic_field_basics.py)

### 물리적 배경

#### 비오-사바르 법칙 (Biot-Savart Law)
```
dB = (μ₀/4π) * (I*dl × r̂)/r²
```

전류가 **자기장**을 만듭니다!

#### 무한 직선 전류
```
B = μ₀*I/(2πr)
```

여기서:
- `μ₀ = 4π×10⁻⁷ T·m/A` (진공 투자율)
- `I`: 전류 (A)
- `r`: 전선으로부터 거리

### 주요 개념

1. **오른손 법칙**
   - 엄지: 전류 방향
   - 나머지 손가락: 자기장 방향 (원형)

2. **자기장의 특징**
   - 항상 **원형** (closed loop)
   - **자기 홀극 없음**: ∇·B = 0
   - 전류에 수직

3. **앙페르 법칙**
   ```
   ∮ B·dl = μ₀*I_enclosed
   ```

### 물리적 통찰

**전기장 vs 자기장**:

| 전기장 | 자기장 |
|-------|-------|
| 전하가 근원 | 전류(움직이는 전하)가 근원 |
| 방사형 | 원형 |
| 홀극 존재 | 홀극 없음 |
| ∇·E ≠ 0 | ∇·B = 0 |

**왜 원형인가?**
- 로렌츠 변환 (상대성 이론)
- 움직이는 전하 → 자기장
- 원형 대칭성

## 🔬 Program 05: 로렌츠 힘 (05_lorentz_force.py)

### 물리적 배경

#### 로렌츠 힘 (Lorentz Force)
```
F = q(E + v×B)
```

하전 입자가 전자기장에서 받는 힘!

#### 사이클로트론 운동
- 자기장에 수직으로 입사 → **원운동**
- 주파수: `f = qB/(2πm)`
- 반지름: `r = mv/(qB)`

#### E×B 표류 (Drift)
- 전기장과 자기장이 교차하면 **표류**
- 표류 속도: `v_drift = E/B`

### 주요 개념

1. **자기력의 특징**
   - 속도에 **수직**
   - **일을 하지 않음**: `dW = F·dl = 0`
   - 운동 에너지 불변

2. **응용**
   - **사이클로트론 가속기**: 입자 가속
   - **질량 분석기**: 질량 측정
   - **자기권**: 지구 자기장이 우주선 차단

3. **수치 방법**
   - **RK4**: 운동 방정식 풀이
   - 정확도 높음

### 물리적 통찰

**왜 원운동인가?**
```
F = qvB = mv²/r
→ r = mv/(qB)
```

자기력이 구심력!

**Van Allen Belt**:
- 지구 자기장에 갇힌 하전 입자
- 나선 운동 + 표류
- 극지방에서 대기와 충돌 → 오로라

## 🔬 Program 06: 1D 파동 방정식 (06_maxwell_1d.py)

### 물리적 배경

#### 파동 방정식
```
∂²E/∂t² = c²∂²E/∂x²
```

맥스웰 방정식의 해!

#### FDTD (Finite Difference Time Domain)
- 시간과 공간을 격자로 나눔
- 유한 차분법으로 미분 근사
- 명시적(explicit) 방법

### 주요 개념

1. **CFL 조건 (Courant-Friedrichs-Lewy)**
   ```
   c*dt/dx ≤ 1
   ```
   안정성을 위한 필수 조건!

2. **유한 차분 근사**
   ```
   ∂²E/∂x² ≈ (E[i+1] - 2E[i] + E[i-1])/dx²
   ```

3. **경계 조건**
   - 완전 반사: `E[0] = 0`
   - 흡수 경계 (PML): 반사 최소화

### 수치 방법 상세

#### FDTD 알고리즘
```python
E_new[i] = 2*E[i] - E_old[i] + (c*dt/dx)^2 * (E[i+1] - 2*E[i] + E[i-1])
```

**3개의 시간 레벨**:
- `E_old`: 이전 시간 (t - dt)
- `E`: 현재 시간 (t)
- `E_new`: 다음 시간 (t + dt)

**왜 3개가 필요한가?**
- 2차 시간 미분 때문
- Leapfrog 방법

### 물리적 통찰

**파동의 전파**:
- 속도 `c = 3×10⁸ m/s`
- 분산 없음 (모든 주파수 같은 속도)
- 반사와 투과

**CFL 조건의 의미**:
- 수치적 정보 전달 속도 ≥ 물리적 파동 속도
- 안 지키면 **불안정** (발산!)

## 🔬 Program 07: 2D 파동 방정식 (07_maxwell_2d.py)

### 물리적 배경

#### 2D 파동 방정식
```
∂²E/∂t² = c²(∂²E/∂x² + ∂²E/∂y²) = c²∇²E
```

#### 원형 파면 (Circular Wave Front)
- 점 소스 → 동심원 파면
- Huygens 원리

### 주요 개념

1. **2D CFL 조건**
   ```
   c*dt ≤ 1/√(1/dx² + 1/dy²)
   ```
   1D보다 더 엄격!

2. **회절 (Diffraction)**
   - 장애물을 만나면 **휘어짐**
   - Huygens-Fresnel 원리
   - 슬릿 실험

3. **간섭 (Interference)**
   - 두 파동의 중첩
   - 보강 간섭: 위상 같음
   - 상쇄 간섭: 위상 반대

### 수치 방법 상세

#### 2D Laplacian
```python
∇²E = (E[j,i+1] - 2E[j,i] + E[j,i-1])/dx² + 
      (E[j+1,i] - 2E[j,i] + E[j-1,i])/dy²
```

**5점 Stencil**:
```
     E[j-1,i]
        |
E[j,i-1] - E[j,i] - E[j,i+1]
        |
     E[j+1,i]
```

### 물리적 통찰

**회절의 중요성**:
- 빛의 파동성 증명
- 광학 소자 설계
- X선 회절 (결정 구조 분석)

**계산 복잡도**:
- 1D: O(Nx × Nt)
- 2D: O(Nx × Ny × Nt)
- 3D: O(Nx × Ny × Nz × Nt) → 매우 무거움!

## 🔬 Program 08: 다중 점전하 (08_multiple_charges.py)

### 물리적 배경

#### 중첩 원리 (Superposition Principle)
```
E_total = E₁ + E₂ + E₃ + ...
V_total = V₁ + V₂ + V₃ + ...
```

선형성 덕분에 **합**이 가능!

### 주요 개념

1. **사극자 (Quadrupole)**
   - 배치: `+q, -2q, +q`
   - 멀리서: `E ∝ 1/r³`
   - 쌍극자보다 빠르게 감소

2. **육극자 (Hexapole)**
   - 6개 전하 + 중심 전하
   - 대칭성
   - 이온 트랩 응용

3. **분자 모델**
   - 실제 분자의 전하 분포
   - 쌍극자 모멘트
   - 분자간 힘

### 물리적 통찰

**다극자 전개 (Multipole Expansion)**:
```
V(r) = Q/r + p·r̂/r² + ...
```

- 홀극(monopole): `1/r`
- 쌍극자(dipole): `1/r²`
- 사극자(quadrupole): `1/r³`

**왜 중요한가?**
- 분자 동역학
- 단백질 구조
- 정전기 상호작용

## 🔬 Program 09: 전자기파 애니메이션 (09_em_wave_animation.py)

### 물리적 배경

#### 평면파 해
```
E = E₀ sin(kz - ωt) ŷ
B = B₀ sin(kz - ωt) x̂
```

여기서:
- `k = 2π/λ` (파수)
- `ω = 2πf` (각진동수)
- `E₀/B₀ = c`

#### E ⊥ B ⊥ k
- 전기장, 자기장, 전파 방향이 **서로 수직**
- 횡파 (transverse wave)

### 주요 개념

1. **포인팅 벡터 (Poynting Vector)**
   ```
   S = (1/μ₀) E × B
   ```
   에너지 흐름의 방향과 크기!

2. **전자기파 스펙트럼**
   - 라디오파: `< 300 MHz`
   - 마이크로파: `300 MHz ~ 300 GHz`
   - 적외선: `300 GHz ~ 430 THz`
   - 가시광선: `430 ~ 750 THz`
   - 자외선, X선, 감마선: `> 750 THz`

3. **편광 (Polarization)**
   - E 필드의 진동 방향
   - 선편광, 원편광, 타원편광

### 물리적 통찰

**E₀/B₀ = c의 의미**:
```
c = 1/√(μ₀ε₀)
```

진공의 전자기적 성질이 광속을 결정!

**에너지 밀도**:
```
u = ε₀E² = B²/μ₀
```

전기장과 자기장이 **같은 에너지**를 가짐!

## 🔬 Program 10: 도체 전위 분포 (10_conductor_potential.py)

### 물리적 배경

#### 라플라스 방정식
```
∇²V = 0
```

전하가 없는 영역에서의 전위!

#### 가우스-자이델 반복법
```
V[i,j] = (V[i+1,j] + V[i-1,j] + V[i,j+1] + V[i,j-1]) / 4
```

주변 4점의 평균!

### 주요 개념

1. **경계 조건 (Boundary Conditions)**
   - **디리클레 (Dirichlet)**: 전위 고정
   - **노이만 (Neumann)**: 전기장 수직 성분 고정

2. **수렴 (Convergence)**
   ```
   ||V_new - V_old|| < tolerance
   ```

3. **평행판 커패시터**
   - 이론해: `E = V/d` (균일)
   - 수치해와 정확히 일치!

### 수치 방법 상세

#### 5점 Stencil
```
∇²V ≈ (V[i+1,j] + V[i-1,j] + V[i,j+1] + V[i,j-1] - 4V[i,j])/h²
```

**유도**:
```
∂²V/∂x² ≈ (V[i+1,j] - 2V[i,j] + V[i-1,j])/dx²
∂²V/∂y² ≈ (V[i,j+1] - 2V[i,j] + V[i,j-1])/dy²
```

#### 반복법 비교

| 방법 | 수렴 속도 | 복잡도 |
|------|----------|--------|
| Jacobi | 느림 | 간단 |
| Gauss-Seidel | 중간 | 중간 |
| SOR | 빠름 | 복잡 |
| Multigrid | 매우 빠름 | 매우 복잡 |

### 물리적 통찰

**원통 커패시터**:
```
V(r) = V₀ ln(r_outer/r) / ln(r_outer/r_inner)
E(r) = V₀ / (r ln(r_outer/r_inner))
```

**커패시턴스**:
```
C = Q/V = 2πε₀L / ln(r_outer/r_inner)
```

**응용**:
- 동축 케이블
- 커패시터 설계
- 반도체 소자

## 📐 수치 방법 심화

### FDTD 알고리즘

#### 안정성 (CFL 조건)
```
1D: c*dt/dx ≤ 1
2D: c*dt ≤ 1/√(1/dx² + 1/dy²)
3D: c*dt ≤ 1/√(1/dx² + 1/dy² + 1/dz²)
```

**의미**: 수치 정보 전달 속도 ≥ 물리 파동 속도

#### 정확도
- **시간**: 2차 정확도 (O(dt²))
- **공간**: 2차 정확도 (O(dx²))

### 경계 조건 처리

1. **완전 반사 (Perfect Conductor)**
   ```
   E_tangential = 0
   ```

2. **흡수 경계 (Absorbing BC)**
   - Mur ABC (1차)
   - Liao ABC (2차)
   - PML (Perfectly Matched Layer) - 최고

3. **주기 경계 (Periodic BC)**
   ```
   E[0] = E[N-1]
   ```

### 반복법 수렴

#### 수렴 조건
```
ρ(G) < 1
```

여기서 `ρ(G)`는 반복 행렬의 spectral radius

#### 가속 기법
- **SOR (Successive Over-Relaxation)**
  ```
  V_new = (1-ω)*V_old + ω*V_GS
  ```
  
- **Multigrid**
  - 여러 격자 레벨 사용
  - 저주파 오차 빠르게 제거

## ❓ 자주 묻는 질문

### Q1: CFL 조건을 안 지키면 어떻게 되나요?
**A**: 수치 해가 **발산**합니다! 파동이 격자를 "뛰어넘어" 정보가 손실됩니다.

### Q2: 왜 전기장은 1/r², 전위는 1/r인가요?
**A**: 전기장은 **벡터**이고, 전위는 **스칼라**입니다. 전위를 미분하면(기울기) 전기장이 되므로, 차수가 하나 높습니다.

### Q3: 자기 홀극은 왜 없나요?
**A**: 실험적으로 발견된 적이 없습니다. 이론적으로는 존재 가능하지만 (GUT 이론), 아직 관측 안 됨.

### Q4: 광속이 왜 c = 1/√(μ₀ε₀)인가요?
**A**: 맥스웰 방정식에서 유도됩니다. 진공의 **전자기적 성질**이 광속을 결정!

### Q5: FDTD vs FEM, 어느 것이 좋나요?
**A**: 
- **FDTD**: 시간 의존, 광대역, 구현 간단, 직교 격자
- **FEM**: 정적, 복잡한 형상, 정확도 높음, 임의 격자

### Q6: Python으로 실제 전자기 시뮬레이션이 가능한가요?
**A**: 
- **학습/프로토타입**: 충분히 가능!
- **실제 엔지니어링**: C++/Fortran (속도), 상용 소프트웨어 (CST, HFSS, COMSOL)

### Q7: 프로그램이 너무 느려요!
**A**: 
1. 격자를 성글게 (Nx, Ny 줄이기)
2. NumPy 벡터화 사용
3. Numba JIT 컴파일
4. C++로 재작성
5. GPU 사용 (CUDA, OpenCL)
